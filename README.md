## 项目简介

企业级智能库存管理系统是一套完整的库存管理解决方案，采用前后端分离架构，基于主流技术栈开发。系统涵盖了企业库存管理的核心业务场景，包括基础档案管理、采购管理、销售管理、库存管理、系统权限管理等模块。

### 核心亮点

- **完整的RBAC权限模型** - 用户-角色-菜单三级权限控制，支持按钮级权限
- **多仓库支持** - 支持多仓库、多库位、多批次库存管理
- **全流程订单管理** - 采购/销售订单全生命周期管理，支持审核流程
- **实时库存监控** - 实时库存查询、库存预警、库存流水追踪
- **数据可视化** - 丰富的图表展示，直观的数据分析
- **操作审计** - 完整的操作日志记录，便于追溯

## 功能特性

### 系统管理

- **用户管理** - 用户增删改查、重置密码、分配角色、状态管理
- **角色管理** - 角色配置、菜单权限分配、数据权限设置
- **菜单管理** - 菜单配置、权限标识、按钮权限
- **部门管理** - 部门树形结构、部门信息维护
- **操作日志** - 操作记录查询、日志详情展示

### 基础档案

- **商品管理** - SPU/SKU管理、规格配置、价格设置
- **仓库管理** - 多仓库支持、仓库信息维护
- **供应商管理** - 供应商档案、联系方式管理
- **客户管理** - 客户档案、联系方式管理

### 采购管理

- **采购订单** - 订单创建、审核、查询、导出
- **采购入库** - 入库确认、库存自动更新

### 销售管理

- **销售订单** - 订单创建、审核、查询、导出
- **销售出库** - 出库确认、库存自动扣减

### 库存管理

- **实时库存** - 多维度库存查询、库存预警
- **库存流水** - 出入库记录、流水查询
- **库存盘点** - 盘点单创建、差异处理

### 系统监控

- **在线用户** - 在线用户监控、强制下线
- **操作日志** - 系统操作记录查询
- **数据监控** - 首页数据可视化展示

## 快速开始

### 环境准备

确保本地已安装以下环境：

- JDK 1.8+
- Maven 3.6+
- Node.js 14+
- MySQL 8.0+

### 后端启动

```bash
# 1. 克隆项目
git clone <repository-url>
cd inventory-v2/inventory-backend

# 2. 导入数据库
mysql -u root -p < sql/inventory_v2.sql

# 3. 修改数据库配置
# 编辑 src/main/resources/application.yml
# 修改数据库连接信息

# 4. 启动后端服务
mvn spring-boot:run

# 后端服务地址: http://localhost:8080/api
# API文档地址: http://localhost:8080/api/doc.html
```

### 前端启动

```bash
# 1. 进入前端目录
cd inventory-v2/inventory-frontend

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run serve

# 前端访问地址: http://localhost:8081
```

### 登录系统

启动完成后访问：http://localhost:8081

默认管理员账号：
- 用户名：`admin`
- 密码：`123456`
